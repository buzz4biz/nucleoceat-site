# Núcleo CEAT - Site Institucional

Site institucional do Núcleo de Comunicação Estratégica Avançada para Terapeutas.

## Tecnologias

- HTML5
- CSS3 (Design System personalizado)
- JavaScript Vanilla
- Tipografia: Crimson Pro + Work Sans

## Deploy

Site hospedado no Render.

## Estrutura

- `index.html` - Página principal
- `blog.html` - Página de publicações

## Design

Design editorial institucional de alto nível, inspirado em institutos de pesquisa e publicações acadêmicas de prestígio.
